# Sorting Algorithms

* Bubble Sort
* Bucket Sort
* Counting Sort
* Heap Sort
* Merge Sort
* Quick Sort
* Radix Sort
* Selection Sort

## Properties of Sorting algorithms

| **Sorting Algorithm** | **Time Complexity** | **Space Complexity** | **Where to Use** |
| --------------------- |:-------------------:|:--------------------:|:----------------:|
| Bubble Sort | O(n^2) | - | - |
| Bucket Sort | O(n) | - | - |
| Counting Sort | O(n) | - | - |
| Heap Sort | O(Log(n)) | - | - |
| Merge Sort | O(nLog(n)) | - | - |
| Quick Sort | O(nLog(n)) | - | - |
| Radix Sort | O(n) | - | - |
| Selection Sort | O(n^2) | - | - |
